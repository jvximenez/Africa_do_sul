# África do Sul — roteiro 2026

Site estático com mapa interativo (Leaflet + OpenStreetMap): dias, fases e pontos da viagem.

**Abrir no navegador:** [jvximenez.github.io/Africa_do_sul](https://jvximenez.github.io/Africa_do_sul/)

**Editar:** altere `index.html` (dados em `DAYS`, fotos no campo `img` via `wc('Nome exato do ficheiro.jpg')` — títulos reais no [Wikimedia Commons](https://commons.wikimedia.org); `Special:FilePath` exige o nome da página do ficheiro, não um nome inventado). Depois `git add -A && git commit -m "…" && git push`.
