# África do Sul — roteiro 2026

Site estático com mapa interativo (Leaflet + OpenStreetMap): dias, fases e pontos da viagem.

**Abrir no navegador:** [jvximenez.github.io/Africa_do_sul](https://jvximenez.github.io/Africa_do_sul/)

**Editar:** altere `index.html` (dados em `DAYS`, fotos no campo `img` de cada ponto — hoje usa [Lorem Picsum](https://picsum.photos) com seeds fixas; pode trocar por URLs do seu álbum ou Wikimedia), depois `git add -A && git commit -m "…" && git push`.
